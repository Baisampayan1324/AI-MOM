Google Meet Integration Guide

🎉 Enhanced Permission System

The extension now automatically detects and inherits microphone permissions from Google Meet!

1Install the Extension

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (top right toggle)
  3. Click "Load unpacked"
  4. Select the p:\\AI_MOM\\browser-extension folder
  5. Verify the extension appears in your toolbar

2Start the Backend

cd backend
python main.py

Ensure the backend is running on localhost:8000

3Open Google Meet

  1. Go to meet.google.com
  2. Join or start a meeting
  3. Allow microphone access when prompted by Google Meet
  4. This is crucial - the extension will inherit this permission!

4Use the Extension

  1. Click the AI Meeting Extension icon in your browser toolbar
  2. The extension will automatically check permissions
  3. If Google Meet has microphone access, the extension inherits it
  4. Click "Open Transcription Panel"
  5. Click "Start Recording" in Online Mode
  6. Speak and watch real-time transcription!

⚠️ Important Notes

  • The extension must be used on the same Google Meet tab where microphone permission was granted
  • If you get permission errors, click "Check Permissions" in the extension popup
  • Refresh the Google Meet page if the extension doesn't work initially
  • Ensure the backend is running before starting recording
